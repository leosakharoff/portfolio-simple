Spring Radio

Programming

Spring Radio is a community-driven online radio project built with local artists and friends. It's a platform for experimentation, collaboration, and creating shared sonic experiences.

The project features a full livestream platform with scheduling capabilities, cloud storage for archiving broadcasts, and LibreTime integration for automated programming. It serves as a gathering place for the community—a digital space where artists can share their work, experiment with live formats, and connect with listeners.

Beyond the technical infrastructure, Spring Radio represents a commitment to local creative networks and the idea that technology can facilitate genuine artistic collaboration and community building.

Technologies / Tools

LibreTime, Cloud Storage, Web Development, Streaming Infrastructure
